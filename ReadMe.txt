Для проверки Web Api использовать проект WebApi2. В PostMan создать запрос с параметрами username=admin@gmail.com password=12345.
Из результата этого запроса извлечь токен и создать новый запрос с авторизацией Bearer и данным токеном. В данном запросе задать параметр
sId  с  id рекламного блока. Отправить запрос и получить ответ.Время жизни токена 10 мин.
